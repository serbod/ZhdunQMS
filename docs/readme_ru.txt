= Queue Management System =

== Особенности ==

* Бесплатно
* Не требуется редкое и дорогое оборудование


== Дисплей, терминал ==

- Получить билет можно нажимая кнопку/педаль, подключенную к звуковой карте или GPIO
* Получаемый номер билета отображается на экране, его можно записать или запомнить самому
- Рядом с номером билета отображается простая картинка с предметом, которую легко запомнить
- Управлять можно одной кнопкой, коротким и долгим нажатием
- Доска объявлений
* Звуки, голосовые сообщения


== Рабочее место оператора ==

* Значок в трее, меню действий
* Плавающая панель
- Настройка расписания (рабочего времени)
- Статистика по билетам и времени
* Санитарный перерыв 15 мин
- Объявление
- Ограничение числа билетов
* Запись логов


== Рабочее место администратора ==

* Несколько дисплеев/терминалов
- Аннулирование билетов
- Резервирование билетов на заданное время



== OS ==

* Windows
